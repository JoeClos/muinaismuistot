# muinaismuistot

Nettisivu Suomen muinaismuistojen näyttämiseen kartalla.

Tekniikkana on selaimessa pyörivä [OpenLayers](http://openlayers.org/)-kartta. Palvelulla ei ole omia palvelitoiminnallisuuksia vaan kaikki data haetaan dynaamisesti avoimista rajapinnoista.

Sivusto käyttää pohjakarttana [Maanmittauslaitoksen](https://www.maanmittauslaitos.fi/kartat-ja-paikkatieto/asiantuntevalle-kayttajalle/kartta-ja-paikkatietojen-rajapintapalvelut-17) avoimesta WMTS-palvelusta saatavaa tausta-, ortokuva- ja maastokarttaa ([Creative Commons CC By 4.0 -lisenssillä](https://creativecommons.org/licenses/by/4.0/)).

Muinaisjäännösten tiedot haetaan [Museoviraston](https://www.avoindata.fi/data/fi/dataset/museoviraston-kulttuuriymparistoaineistot-suojellut-kohteet-wms-palvelu) avoimesta ArcGIS-palvelusta ([Creative Commons CC By 4.0 -lisenssillä](https://creativecommons.org/licenses/by/4.0/)).

Ahvenanmaan maakuntahallinnon rekisterin tiedot haetaan [avoimen rajapinnan](https://www.kartor.ax/datasets/fornminnen) kautta ([Creative Commons CC By 4.0 -lisenssillä](https://creativecommons.org/licenses/by/4.0/)).

Sivusto: http://muinaismuistot.info
